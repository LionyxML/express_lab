# Example Express SSR Site

### Start project

Install the dependencies:
`npm install`

Set the

Run the server:
`node index.js`

or

`NODE_ENV=production node index.js`

Access in your browser:
`http://localhost:3000`

### Testing

#### Run all tests

`npm test`

#### Run tests in watch mode with menu

`npm test -- --watch`

#### Generate coverage report

`npm test -- --coverage`
